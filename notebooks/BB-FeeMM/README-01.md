# 分析概要
- BitbankはMaker約定の際にFeeが(2bps)得られる取引所です. このFee狙いの高頻度MM戦略を作りたいと思っています.
- この戦略では価格変動による収益はトントンでよいので、なるべくmaker約定をさせてFeeを稼ぎます (もちろん狙えるなら価格変動やスプレッドも取りたいです).
- 一方でtaker約定の際のFeeは12bpsと大きいので注意してください.
- 他の取引所の約定や板の動きも適宜使ってください.
- また、bitbank自体の板の動きが極短期の価格変動に影響を与えることが知られていますのでこれを活用してもよいかもしれません

# データソース
venue = bitbankの各通貨ペア. 約定寮を稼ぐためにメジャーな通貨から取り組むと良い.
