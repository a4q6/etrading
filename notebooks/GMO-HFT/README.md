# 分析概要
venue=GMOでのレバレッジ取引(sym=XXXJPY)においてはmaker/taker共に取引手数料が0bpsであり高頻度戦略の構築の余地がある.
1日あたりの取引回数>500程度のMarket Making戦略を構築して安定した絶対収益を狙いたい.
目標としては月次PLが絶対額で+50~100%, sharpe ratio > 1.5.
まずは既存のレポジトリの戦略にはとらわれず、自由な発想で戦略構築、検証を行って欲しい.
厳密なbacktestの実行方法については `@notebooks/Dev-MM-backtest.ipynb` が参考になる(しかし、これにとらわれる必要もない).

# データソース
- venue=GMO, sym=xxxJPYが取引対象. 
- 特徴量としては他のexchangeの値動きや板の動き、取引量などを使うことも考えられるため、まずは有効な特徴量の検証からはじめてみてほしい
- backtestに関しては最低1ヶ月以上をtest期間に設定して欲しい
- 非常に大量のデータになるため、メモリリークに注意. 長期の統計的性質をEDAする際にはLoad -> resample してから分析した方がよい. Backtestについては1weekずつ(Sun-Sat)実行するのが良い.
