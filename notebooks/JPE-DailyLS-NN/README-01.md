# 分析概要
- 日本株式の日次のロングショートのポジションを取ることで絶対収益を追求する戦略を構築したいと思っています。
- 執行はBacktest結果の再現性をたかめるため、Daily Open -> Daily Closeで行います. 
- Short可能な銘柄に絞るために、銘柄は大型・中型のみに絞ります.
- 業種の制限は設けません. 予測の効率性の高いものを用いて結構です.
- TOPIXを上回る絶対収益を期待します.
- モデルの構築にあたってはNeural Network(torch)のモデルを使いたいです. モデルのアーキテクチャについてはお任せします.
- ターゲットは分析設定により与えられたユニバースのOpen2Closeのリターンのランクです. モデルのシグナルに従い上位をロングし下位をショートすることで差分を収益化することを目指します.

# データソース
JQuantsAPIから取得可能なデータが `etr.data.jquants.sqlite.get(query)`でロードできます.
テーブル一覧についてはDDLファイルを参照してください.
個別株およびTOPIX指数のDaily OHLCVは2016年から約10年分あります.
他にも有用そうなデータがあればリークに注意しつつ適宜用いてください.
